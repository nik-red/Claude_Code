## Usage

## Step 1: Pick the python environment
Before using this skill, make sure you have the required python environment set up. You can run/install the dependencies using my '.venv' environment which is located at "C:\ClaudeCode\.venv"

## Step 2: Run the Python Script To Visualize Data & Generate Visualizations
- You need to run the Python script located at ".claude/skills/visualize/scripts/visualize.py" to visualize the data and generate visualizations. This script will generate visualizations based on the data you have and save them in the appropriate directory.
